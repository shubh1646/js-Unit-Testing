# js-Unit-Testing
Using Mocha, Chai &amp; nyc for testing coverage. Created for learning purpose.



# to run test  
 - npm run test 

 # to run cover
 - npm run cover 
 - in package.json under script include  "cover" : "nyc --reporter=html --reporter=text mocha"
